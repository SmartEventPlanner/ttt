# 🎉 スマートイベントプランナー

**AIがあなたのイベント企画をスマートにアシスト！**  
忙しくても、センス良く、手軽にイベントを開きたいあなたへ。

---

## 🧠 コンセプト（5W1H）

### 👩‍💼 Who（誰が）
- 友達と集まりたい**20代の社会人女性**

### 📌 What（何を）
- **AIアシストのイベント企画Webツール**
  - 日程調整  
  - 会場・ケータリングの提案  
  - 招待チェックリストの作成  

### 🕒 When（いつ）
- 仕事終わりや週末に**予定を立てたい時**

### 🌍 Where（どこで）
- **オンライン**（スマホやPCからどこでも利用可能）

### 💡 Why（なぜ）
- 忙しい毎日でも、**イベントを簡単＆おしゃれに開きたい！**  
- **時間や手間をかけずに、センスある企画**を実現できるから

### 🚀 How（どうやって）
1. サイトにアクセス  
2. イベント作成を開始  
3. **AIが自動でプランを提案**  
4. 招待を送信  
5. 企画から管理まで**すべて一括で完了！**
