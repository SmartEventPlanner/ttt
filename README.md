# スマートイベントプランナー

## 🧑‍🤝‍🧑 Who（誰が）

- 友達と集まりたい20代の社会人女性

## 💡 What（何を）

- AIがアシストするイベント企画WEBツール
- 日程調整、会場・ケータリングの提案、招待チェックリストなどを提供

## 🕒 When（いつ）

- 仕事終わりや週末に予定を立てたい時

## 🌐 Where（どこで）

- オンライン（スマホやPCからどこでも利用可）

## ❓ Why（なぜ）

- 忙しい毎日でも、イベントを簡単＆おしゃれに開きたいから
- 時間や手間をかけずに、センスある企画ができると嬉しい

## ⚙️ How（どうやって）

1. サイトにアクセス  
2. イベント作成  
3. AIが自動提案  
4. 招待送信  
5. 管理まで一括で完了！
